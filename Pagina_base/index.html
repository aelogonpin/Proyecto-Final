Servicios

Pulsa aqui para accerder al Servicio KVM

Seguro & Estrategico

Este sistema utiliza un protocolo mediante ssh que nos permite utilizar keys para poder acceder a los sistemas o mediante un interfaz incorporado en el momento de creacion de la maquina.

Web & Interface

Este interfaz permite realizar interacciones con el servidor mediante botones que realizan la mayoria de las acciones con la mayor explicación y brevedad posible.

Usuarios & Roles

Los usuarios para cualquier acción del sistema tendrán que tener cuentas con permisos y roles especificos que le permitiran tener acceso a lo que los admnistrador crean convenientes.

CMS & eCommerce

Este proyecto tambien se centra en servicios de CMS basado en Wordpress o Owncloud que nos dará vida.
